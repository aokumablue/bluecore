# パフォーマンス標準

## メモ化・キャッシュ

- 高コストな計算は結果を再利用する
- 依存値が変わらない限り再計算しない
- キャッシュの無効化条件を明確にする

## 遅延読み込み

- 重要でないリソースは遅延読み込みする
- 初期表示・起動を重くしない
- 読み込み中の状態を示す

## DB クエリ

- 必要なカラムだけ取得する
- LIMIT とインデックスを活用する
- N+1 問題を避ける
